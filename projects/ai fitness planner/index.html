AI Fitness & Nutrition Planner

Personalised weekly workout plans, meals, and shopping lists powered by GPT-4o-mini.

Python Streamlit OpenAI

Features

  • Log daily fitness & nutrition metrics (weight, steps, sleep, macros)
  • Set personal preferences (goals, diet type, calorie targets, allergies)
  • Generate AI-powered weekly plans with structured workouts and meal plans
  • Auto-generated shopping list grouped by category
  • Manual JSON paste fallback if you prefer using your own AI

How to run

This is a Python app that runs locally with Streamlit:

cd "projects/ai fitness planner" pip install -r requirements.txt streamlit run app.py

Then open http://localhost:8501 in your browser.

Setup

Create a file at .streamlit/secrets.toml with your OpenAI key:

OPENAI_API_KEY = "sk-..."
